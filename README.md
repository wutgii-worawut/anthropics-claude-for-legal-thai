# Claude for Legal — คู่มือภาษาไทย

เอกสารแปลและอธิบายภาษาไทยของ [`anthropics/claude-for-legal`](https://github.com/anthropics/claude-for-legal) — ครอบคลุมทุกโฟลเดอร์ ทุก plugin ทุก skill

**🌐 อ่านออนไลน์**: https://wutgii-worawut.github.io/anthropics-claude-for-legal-thai/

## โครงสร้าง

```
.
├── _config.yml                    # Jekyll + just-the-docs config
├── Gemfile                        # Ruby deps
├── index.md                       # หน้าแรก
├── 01-overview/                   # ภาพรวม
├── 02-folders/                    # โครงสร้างทุกโฟลเดอร์
├── 03-plugins/                    # 12 vertical plugins
├── 04-external-plugins/           # CoCounsel + community
├── 05-cookbooks/                  # Managed Agent cookbooks
├── 06-scripts/                    # scripts/
├── 07-references/                 # template files
├── 08-deployment/                 # การ deploy
├── 09-development/                # การ dev
├── 10-glossary/                   # อภิธานศัพท์
└── .github/workflows/pages.yml    # build + deploy
```

## รันที่เครื่อง (Local Preview)

```bash
bundle install
bundle exec jekyll serve
# เปิด http://localhost:4000/anthropics-claude-for-legal-thai/
```
